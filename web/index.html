Get your Gmail data into stats.db one of two ways, running the command from the repo root in a terminal. Once a run starts, this page notices it and fills in live.

Scan Gmail (OAuth)

Reads message metadata over the Gmail API. Needs a Google Cloud OAuth client (credentials.json in the repo root) plus a one-time browser consent.

cargo run

Using Advanced Protection? Google blocks this method outright (Error 400: policy_enforced) — use the Takeout import instead.

Import a Google Takeout mailbox

  1. At takeout.google.com, export only Mail (mbox format).
  2. Wait for the export email — Google typically takes about 2 days to prepare it for Advanced Protection accounts.
  3. Download, unzip, and import the .mbox file:
cargo run -- import "path/to/All mail Including Spam and Trash.mbox"
